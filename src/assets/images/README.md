# تصاویر بازی

## تصاویر مورد نیاز:

### 1. پس‌زمینه صفحه شروع
**نام فایل:** `splash-bg.jpg`
- اندازه توصیه شده: 1080x1920 پیکسل (عمودی برای موبایل)
- حجم فایل: کمتر از 500KB

### 2. دکمه شروع
**نام فایل:** `btn-start.png`
- اندازه توصیه شده: عرض 300-400px، ارتفاع 80-100px
- فرمت: PNG با شفافیت (transparent background)
- اگر فایل وجود نداشته باشد، دکمه متنی نمایش داده می‌شود

### 3. دکمه ادامه
**نام فایل:** `btn-continue.png`
- اندازه توصیه شده: عرض 300-400px، ارتفاع 80-100px
- فرمت: PNG با شفافیت (transparent background)
- اگر فایل وجود نداشته باشد، دکمه متنی نمایش داده می‌شود

**فرمت‌های پشتیبانی شده:**
- JPG / JPEG
- PNG (توصیه می‌شود برای دکمه‌ها)
- WebP
