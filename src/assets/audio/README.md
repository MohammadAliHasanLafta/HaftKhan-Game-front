# فایل‌های صوتی بازی

## فایل‌های مورد نیاز:

### 1. موسیقی پس‌زمینه صفحه شروع
**نام فایل:** `splash-bg-music.mp3` یا `splash-bg-music.ogg`
- فرمت‌های پشتیبانی: MP3, OGG, WAV
- توصیه: MP3 برای سازگاری بیشتر

### 2. موسیقی پس‌زمینه بازی
**نام فایل:** `game-bg-music.mp3` یا `game-bg-music.ogg`
- فرمت‌های پشتیبانی: MP3, OGG, WAV
- توصیه: MP3 برای سازگاری بیشتر

## نکات:
- فایل‌ها باید حجم مناسبی داشته باشند (کمتر از 2MB توصیه می‌شود)
- موسیقی به صورت loop پخش می‌شود
- کاربر می‌تواند موسیقی را خاموش/روشن کند
