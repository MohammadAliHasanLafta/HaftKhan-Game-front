# فونت‌های بازی

لطفاً فایل فونت TTF خود را در این پوشه قرار دهید.

## نحوه استفاده:

### 1. فایل فونت را اضافه کنید:
- فایل فونت TTF را با نام `game-font.ttf` در این پوشه قرار دهید
- (اختیاری) اگر فونت Bold دارید، آن را با نام `game-font-bold.ttf` قرار دهید

### 2. تنظیمات:
فونت به صورت خودکار در کل پروژه استفاده می‌شود. اگر نام فایل شما متفاوت است، فایل `src/style.css` را ویرایش کنید و نام فایل را تغییر دهید.

## ساختار پیشنهادی:

```
src/assets/fonts/
├── game-font.ttf          (فونت معمولی)
└── game-font-bold.ttf     (فونت Bold - اختیاری)
```

## نکات:
- اگر فقط یک فایل فونت دارید، همان را با نام `game-font.ttf` قرار دهید
- فونت در تمام کامپوننت‌ها به صورت خودکار اعمال می‌شود
- اگر فونت Bold ندارید، خط مربوط به `game-font-bold.ttf` را از `src/style.css` حذف کنید
