# فایل‌های ویدیویی بازی

## فایل‌های مورد نیاز:

### 1. ویدیوی داستان شروع
**نام فایل:** `story-intro.mp4`
- فرمت‌های پشتیبانی: MP4, WebM, OGG
- اندازه توصیه شده: 1080x1920 پیکسل (عمودی برای موبایل)
- مدت زمان: هر مدت که می‌خواهید
- حجم فایل: کمتر از 10MB برای بارگذاری سریع‌تر

## نکات:
- ویدیو به صورت خودکار پخش می‌شود
- بعد از 20 ثانیه، دکمه "رد کردن" نمایش داده می‌شود
- کاربر می‌تواند ویدیو را رد کند یا تا انتها تماشا کند
